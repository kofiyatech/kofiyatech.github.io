<h2>Try Storyteller on PyPI</h2>

  <p>
    We are excited to announce that <strong>STORYTELLER</strong> is now available for testing on
    <a href="https://pypi.org/project/storyteller-dhs/" target="_blank" rel="noopener">PyPI</a>!
  </p>

  <p>
    STORYTELLER is a Python package that makes it easier to explore and work with <strong>Demographic and Health Survey (DHS)</strong> datasets.
    It provides simple commands for launching a web-based data explorer, running predefined queries, and exporting cleaned datasets for further analysis.
  </p>

  <hr>

  <h2>What You Can Do</h2>
  <ul>
    <li>Start an interactive <strong>Datasette web app</strong> to browse DHS databases.</li>
    <li>Run <strong>predefined queries</strong> and export datasets in ready-to-use formats.</li>
    <li>Enable <strong>full-text search (FTS)</strong> on DHS variable tables.</li>
    <li>Export cleaned data to <strong>CSV</strong> and reproducible metadata to <strong>JSON</strong>.</li>
  </ul>

  <hr>

  <h2>How to Install</h2>
  <p>You can install the package directly from PyPI:</p>

  <pre><code>pip install storyteller-dhs</code></pre>
